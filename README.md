# 🛒 Sistema de Análise de Vendas - SQL Server

Projeto desenvolvido com o objetivo de simular um sistema de vendas de uma loja fictícia utilizando SQL Server, aplicando conceitos de modelagem de dados e análise com consultas SQL.

---

## 📌 Sobre o Projeto

Este projeto foi criado para praticar e demonstrar conhecimentos em:

- Estruturação de banco de dados
- Inserção de dados simulados
- Desenvolvimento de indicadores estratégicos (KPIs)
- Análise de dados utilizando SQL

A base contém registros fictícios de vendas realizadas entre janeiro e maio de 2026.

---

## 🧱 Estrutura do Projeto

O repositório está organizado da seguinte forma:

01_criacao_tabela.sql  
→ Script de criação da tabela `sistema_loja`

02_inserts.sql  
→ Inserção de dados simulados de vendas

03_indicadores.sql  
→ Consultas analíticas e indicadores de desempenho

---

## 📊 Indicadores Desenvolvidos

- ✅ Faturamento total
- ✅ Total de vendas realizadas
- ✅ Ticket médio
- ✅ Produto mais vendido
- ✅ Cliente que mais comprou
- ✅ Faturamento mensal
